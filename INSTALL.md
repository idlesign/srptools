# srptools installation

Python `pip` package is required to install `srptools`.


## From sources

Use the following command line to install `srptools` from sources directory (containing `pyproject.toml`):

    pip install .


## From PyPI

Alternatively you can install `srptools` from PyPI:

    pip install srptools

    ; Install with Click package for CLI:
    pip install srptools[cli]


Use `-U` flag for upgrade:

    pip install -U srptools
